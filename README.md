## Описание проекта

Этот проект — Telegram-бот на базе библиотеки `aiogram` для учёта задач и времени.  
Бот позволяет добавлять задачи, действия и прерывания, отслеживать их длительность и получать сводный отчёт за день.

Основные сущности:
- **Задача** — основная работа.
- **Действие** — сопутствующее действие.
- **Прерывание** — отвлечение/перерыв.

Все данные хранятся в базе данных `SQLite` в файле `my_database.db`.

## Технологии

- **Язык**: Python
- **Библиотека бота**: `aiogram`
- **Хранение настроек**: `python-dotenv` (`.env`)
- **База данных**: `SQLite` (модуль `sqlite3`)

Список зависимостей прописан в файле `requirements.txt`.

## Структура проекта

- `main.py` — точка входа, запуск бота и диспетчера.
- `config.py` — загрузка переменных окружения, получение `TOKEN`.
- `database.py` — создание и работа с базой данных (пользователи, задачи, статусы, отчёты).
- `handlers/commands.py` — основные хэндлеры сообщений и логика состояний (FSM).
- `handlers/keyboards.py` — клавиатуры и меню Telegram-бота.
- `requirements.txt` — зависимости Python.
- `Procfile` — конфигурация для деплоймента (например, на Heroku).

## Подготовка и установка

1. **Клонируйте репозиторий** (или скачайте проект):
   ```bash
   git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
   cd TgBot
   ```

2. **Создайте и активируйте виртуальное окружение (рекомендуется)**:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```

3. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Создайте файл `.env` в корне проекта** (если его ещё нет) и пропишите токен бота:
   ```env
   BOT_TOKEN=ваш_telegram_bot_token
   ```

Токен можно получить у `@BotFather` в Telegram.

## Запуск бота локально

После установки зависимостей и настройки `.env` выполните:

```bash
python main.py
```

При первом запуске:
- создаётся база данных `my_database.db`,
- инициализируются таблицы и типы задач (`task`, `action`, `interrupt`).

## Основной функционал бота

- **Добавление задачи**: бот запрашивает название и тип задачи (Задача / Действие / Прерывание), сохраняет её в БД и сразу запускает.
- **Список задач**: выводит все задачи пользователя с указанием выполненных.
- **Учёт времени**: фиксирует время старта/паузы/остановки задач и считает длительность.
- **Отчёт за день**: по кнопке формирует отчёт по задачам за текущий день с удобным форматированием времени.
- **Возобновление задачи**: можно возобновить последнюю приостановленную задачу.
- **Завершение активной задачи**: завершение текущей активной задачи.

Управление ботом происходит через кнопки клавиатуры:
- меню добавления задач,
- основное меню (список задач, отчёт, возврат к добавлению, возобновление задачи).

## Разработка и доработка

- Логику бота и обработчики сообщений можно расширять в файле `handlers/commands.py`.
- Новые кнопки и меню добавляются в `handlers/keyboards.py`.
- Структуру БД и запросы можно изменять в `database.py` (при этом не забывать про миграцию данных, если БД уже используется).

## Требования

- Python 3.10+ (рекомендуется)
- Установленные зависимости из `requirements.txt`
- Действующий Telegram Bot Token в переменной `BOT_TOKEN`
